API Reference

All icons share the same inputs for sizing, accessibility, motion, color and framed UI states.

Input Type Default Description
size 12|14|16|20|24|32 24 Width and height in px
strokeWidth number 2 SVG stroke-width
ariaLabel string | undefined undefined When set, adds role="img"
animate boolean false Enables the icon's semantic CSS animation
tone inherit|foreground|muted|primary|... 'inherit' Uses theme tokens such as --primary and --success
color string | undefined undefined Optional custom override when a theme token is not enough
variant 'outline' | 'filled' 'outline' Filled mode adds a soft currentColor fill while preserving readable strokes
background 'none' | 'soft' | 'solid' 'none' Adds optional host background for badges, toolbars and nav items. Solid backgrounds use theme foreground automatically.
backgroundTone same as tone 'primary' Theme token used by soft and solid backgrounds
backgroundColor string | undefined undefined Optional custom background override
padding number 0 Host padding in px when using a framed icon
radius number | string '0.5rem' Host border radius for background modes
